Installation

phylogeo is dependent on phyloseq and a few other packages. Please see the for installation instructions for phyloseq. After that phylogeo should be easilty installed via devtools: library("devtools") install_github("zachcp", "phyloseq-geo")

Description

The phylosgeo package provides a convenient way to map microbiome-related information. phylogeo relies on the phyloseq package and provdies methods for mappign samples, drawing network diagrams, plotting phylogenetic trees and maps, and correlating physical distacne with ecological distance. or news related to the latest development version, see:
