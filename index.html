Phylogeo

Mapping for Phyloseq

Download this project as a .zip file Download this project as a tar.gz file

Description

The phylogeo package provides a convenient way to map microbiome-related information. phylogeo relies on the phyloseq package and provides methods for mappiin samples, drawing network diagrams, plotting phylogenetic trees and maps, and correlating physical distance with ecological distance.

Installation

phylogeo is dependent on phyloseq and a few other packages. Please see the phyloseq installation instructions for help installing phyloseq.After that phylogeo should be easily installed via the devtools package :

library("devtools")
install_github("phyloseq", "joey711")
install_github("phylogeo","zachcp")
